# Python

## pip

python package installer

### 사용법

* pip list

현재 설치된 패치지 목록을 확인한다.

```bash
pip list
```

* pip install

패키지 설치할 때 사용한다.

```bash
pip install "<package name>"
```

* pip uninstall

패키지 삭제할 때 사용한다.

```bash
pip uninstall "<package name>"
```

## python script 실행

```bash
python *.py
```
